# Noder

#-> A small Quiz game that is designed to run as a node package that can be played on the go. <-

## Set up
1. `just open vsCode`
2. `open terminal`
3. `Do npx noderquiz`
4. `Allow the package to install`
5. `Enjoy the Quiz ♥`
